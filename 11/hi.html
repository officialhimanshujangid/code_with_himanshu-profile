WHAT IS HTML?

HTML stands for Hyper text markup language. It's a markup language that web developers use to structure and describe the content of a webpage (not a programming language). HTML consists of elements that describe different types of content: paragraphs, links, headings, images, video, etc. Web browsers understand HTML and render HTML code as websites.

In HTML, each element is made up of 3 parts:

  1. The opening tag
  2. closing tag
  3. The actual element
